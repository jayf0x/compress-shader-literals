✂️ compress-shader-literals

Strip comments & whitespace from GLSL/WGSL shader literals — try it live, or point it at a real npm package. GitHub · npm

1. Paste & run

Paste JS/TS containing tagged shader literals (glsl\`...\`, wgsl\`...\`) or a raw shader body.

2. Test against a real npm package

Boots a WebContainer in your browser, npm installs the package for real, scans its shipped JS for shader-shaped template literals, and reports before/after size — a lightweight version of this repo's e2e test. Needs a Chromium-based browser.
